XEP-0398 Plugin Readme

Overview

The XEP-0398 plugin adds avatar conversion to Openfire. XEP-0398 is an XMPP standard covering the conversion of avatars between the User Avatar and the vCard-based formats.

Installation

Copy jar into the plugins directory of your Openfire installation. The plugin will then be automatically deployed. To upgrade to a new version, copy the new jar file over the existing file.

Alternatively, download the latest release and upload it via the Openfire Admin Console.

Configuration

The plugin is configured via the Openfire Admin Console. After installation, a new Admin Console page is available. The page can be found on a main menu in the "Server" tab. The name of the page is "Avatarconversion".

Preferences

  • xmpp.xep398.enabled = true|false (default: false)
  • xmpp.xep398.onlypep = true|false (default: false)
  • xmpp.xep0398.shrinkvcardimg = true|false (default: false)Take care: If enabled, the sha hash of the avatar will change!
  • cache.XEP398.maxLifetime = value in millis (default: 3600000 - 1 hour)
  • cache.XEP398.size = value in bytes (default: 20971520 - 20MB)
